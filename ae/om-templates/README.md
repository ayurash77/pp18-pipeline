# AE Output Module Templates

Сюда панель Shotmate экспортирует/отсюда импортирует OM-шаблоны After Effects
(кнопки в шапке списка задач; позже — в настройках панели).

- Файл — это prefs-файл AE `Adobe After Effects <версия>-indep-output.txt`,
  один на каждую версию AE (форматы между версиями не совместимы).
- Экспорт: панель сбрасывает текущие настройки AE на диск и копирует файл сюда.
- Импорт: файл копируется поверх prefs AE и подхватывается без перезапуска
  (`app.preferences.reload()`). Внимание: импорт замещает ВСЕ output-module
  шаблоны текущей версии AE, включая созданные локально после последнего экспорта.
- Синхронизация с GitHub — обычные git commit/push/pull руками.
